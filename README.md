# Sprint Angular - Ford Enter

Projeto desenvolvido para a Sprint Angular da Ford Enter.

## Tecnologias

- Angular 20
- TypeScript
- Bootstrap 5
- Bootstrap Icons
- RxJS
- Angular Router
- Reactive Forms
- HttpClient

## Funcionalidades

- Login
- Home
- Dashboard
- Consulta de veículos
- Consulta por VIN
- Atualização dinâmica das informações
- Integração com API REST

## Estrutura

src/
 ├── core
 │   ├── models
 │   ├── services
 │   └── constants
 │
 ├── pages
 │   ├── login
 │   ├── home
 │   └── dashboard
 │
 └── shared
     ├── header
     └── dashboard-card

## Como executar

npm install

ng serve

API

npm run start

Projeto disponível em:

http://localhost:4200
